# Resources Folder

## Icon Files

### app.ico
Place your custom application icon here. The application will use this for the system tray icon.

If no icon is provided, the application will generate a simple default icon with "SC" text.

### Creating an Icon
You can create an .ico file using:
- Online tools like favicon.io or ico-converter.com
- Windows Paint (save as .ico)
- Professional tools like Adobe Illustrator or GIMP

Recommended size: 32x32 pixels (will be scaled automatically)
